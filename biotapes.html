WHAT IS BIOTAPES

Biotapesâ„¢ is a verified visual baseline protocol used to establish ground-truth evidence for ecological and nature-finance projects.

    It combines:
  • a standardised filming method
  • automatic metadata embedding
  • cryptographic hashing at ingest
  • unique Biotapes Verification IDs
  • lightweight API integration for platforms

Biotapes provides a neutral, independent way to prove what was on the ground, when it was filmed, and that the evidence hasnâ€™t been manipulated.

It is a product of Purpose Built Films, bringing documentary-grade visual integrity into emerging nature markets.

WHY BIOTAPES MATTERS

Nature markets depend on trust - but baselines are often invisible, inconsistent, or unverifiable.

    Biotapes solves this by providing:
  • Transparency - clear, human-visible evidence
  • Consistency - a repeatable filming protocol
  • Security - hash-verified files that cannot be altered
  • Accessibility - easy integration into any platform
  • Neutrality - independent verification trusted by investors, insurers and regulators

Biotapes is designed to support BNG projects, natural capital markets, TNFD-aligned reporting, habitat banks, restoration finance, and NGOs.

HOW BIOTAPES WORKS

Capture

A simple, universal 3-angle filming protocol captures ecological baselines:

  • establishing wide
  • habitat mid-shot
  • macro detail

Metadata

Each clip automatically embeds:

  • GPS
  • timestamp
  • operator ID
  • device ID

Ingest + Hashing

Footage is passed through the Biotapes ingest layer, generating:

  • BiotapesHash_v1 (cryptographic proof)
  • Biotapes Verification ID

Storage

Partners store the file and ID in their systems.

Verification

    Any platform can verify:
  • file integrity
  • metadata
  • authenticity
  • tamper-proof status

Evidence Package

    Data can be exported into a standardised package for:
  • regulators
  • investors
  • insurers
  • ecological consultants
